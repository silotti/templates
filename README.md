# Templates

Biblioteca de templates reutilizáveis para documentos, currículos, relatórios, tabelas e materiais de estudo.

## Estrutura

### curriculos

Modelos de currículos e cartas de apresentação.

### relatorios

Modelos de relatórios técnicos, inspeções, auditorias e documentos diversos.

### seguranca-trabalho

Templates relacionados à Segurança do Trabalho, NR10, NR12, APR, PT, checklists e relatórios.

### tabelas

Planilhas, tabelas de referência e modelos de dados.

## Convenções

* Utilizar Markdown (`.md`) para templates textuais.
* Utilizar DOCX (`.docx`) quando a formatação for importante.
* Utilizar XLSX (`.xlsx`) para planilhas.
* Os templates devem conter apenas a estrutura fixa.
* Os dados variáveis devem ser substituídos no momento da geração.
