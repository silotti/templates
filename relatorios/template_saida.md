# TEMPLATE DE SAÍDA - RELATÓRIOS

## Objetivo

Gerar relatórios seguindo rigorosamente esta estrutura.

## Formato de Saída

A resposta deve conter:

1. Título principal
2. Sumário executivo
3. Desenvolvimento
4. Conclusão
5. Recomendações
6. Anexos (quando aplicável)

## Formatação

* Utilizar Markdown.
* Títulos com "#".
* Subtítulos com "##".
* Tabelas em formato Markdown.
* Não utilizar emojis.
* Linguagem técnica e formal.
